\# Enterprise Data Management \& Automation Engine (EDMAE)



\## Overview

EDMAE is an enterprise-grade automation framework built using advanced VBA architectural principles inside Microsoft Excel.



\## Architectural Design

Layered Architecture:

UI → Application → Domain → Infrastructure



\## Engineering Concepts Demonstrated

\- Interface-based programming

\- Dependency Injection

\- Logging framework

\- Centralized error handling

\- Repository pattern

\- Separation of concerns

\- Testable domain logic



\## Purpose

To demonstrate that VBA can be engineered using professional software architecture standards.
